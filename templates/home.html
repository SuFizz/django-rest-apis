REST API Sample Code

This is a collection of Twitter API samples, covering common use cases such as:

  • Posting a tweet
  • Querying tweets
  • Uploading media
  • Updating a profile photo

These examples also show how to execute the API calls using both Python (via the python-twitter SDK) and Twurl.

python-twitter

The python-twitter library provides a pure Python interface for the Twitter API. It works with Python versions from 2.6+. Python 3 support is under development. python-twitter was started and maintained by Mike Taylor.

You can learn more about python-twitter by visiting the python-twitter Github repo.

Twurl

Twurl is like curl, but tailored specifically for the Twitter API. It knows how to grant an access token to a client application for a specified user and then sign all requests with that access token.

It also provides other development and debugging conveniences such as defining aliases for common requests, as well as support for multiple access tokens to easily switch between different client applications and Twitter accounts.

You can learn more about Twurl by visiting the Twurl Github repo. {% endblock %}
